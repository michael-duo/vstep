## SƠ ĐỒ 2 — Thang điểm & Quy bậc

```
Điểm trung bình 4 kỹ năng (thang 10, làm tròn 0.5)

  0      2      4      6      8     10
  ├──────┼──────┼──────┼──────┼──────┤
                ████████              ← B1 (Bậc 3): 4.0 – 5.5
                       ██████████    ← B2 (Bậc 4): 6.0 – 8.0
                                ████ ← C1 (Bậc 5): 8.5 – 10.0

Cách tính điểm từng kỹ năng:
┌──────────────┬─────────────────────────────────────────┐
│ Listening    │ (Số câu đúng / 35) × 10                 │
│ Reading      │ (Số câu đúng / 40) × 10                 │
│ Writing      │ Giám khảo chấm theo 4 tiêu chí / 10     │
│ Speaking     │ Giám khảo chấm theo 5 tiêu chí / 10     │
└──────────────┴─────────────────────────────────────────┘

Điểm tổng = (L + R + W + S) / 4, làm tròn 0.5

⚠ Chênh 0.1–0.2 điểm = lên hoặc xuống bậc — cần ôn đều 4 kỹ năng!
```

## SƠ ĐỒ 3 — Cấu trúc chi tiết kỹ năng Nghe

```
LISTENING — 40 phút (gồm 7 phút chuyển đáp án)
Nghe MỖI ĐOẠN CHỈ 1 LẦN DUY NHẤT

┌────────────────────────────────────────────────────────────┐
│ PHẦN 1 — 8 câu hỏi                                        │
│ Dạng: Thông báo, hướng dẫn ngắn                           │
│ Độ dài: ~15–20 giây/đoạn                                  │
│ Tip: Có 30 giây đọc câu hỏi trước mỗi đoạn               │
├────────────────────────────────────────────────────────────┤
│ PHẦN 2 — 12 câu hỏi (3 đoạn hội thoại × 4 câu)           │
│ Dạng: Hội thoại giữa 2–3 người                            │
│ Độ dài: 1–3 phút/đoạn                                     │
│ Tip: Chú ý sự thay đổi ý kiến giữa các người nói         │
├────────────────────────────────────────────────────────────┤
│ PHẦN 3 — 15 câu hỏi (3 bài giảng × 5 câu) ← KHÓ NHẤT   │
│ Dạng: Bài giảng học thuật, thuyết trình                   │
│ Độ dài: ~3 phút/bài                                       │
│ Tip: Ghi chú nhanh từ khóa trong lúc nghe                │
└────────────────────────────────────────────────────────────┘

Kỹ năng được đánh giá:
● Nghe thông tin chi tiết        ● Nghe ý chính
● Hiểu ý kiến người nói          ● Suy luận từ ngữ cảnh
● Xác định mục đích người nói
```

## SƠ ĐỒ 4 — Cấu trúc chi tiết kỹ năng Đọc

```
READING — 60 phút · 40 câu trắc nghiệm

┌──────────┬──────────┬──────────┬──────────┐
│  Bài 1   │  Bài 2   │  Bài 3   │  Bài 4   │
│ ~500 từ  │ ~500 từ  │ ~500 từ  │ ~500 từ  │
│ 10 câu   │ 10 câu   │ 10 câu   │ 10 câu   │
│          │          │          │          │
│ Dễ nhất  │          │          │ Khó nhất │
└──────────┴──────────┴──────────┴──────────┘
     ──────────── Độ khó tăng dần ────────────►

Tổng: ~1900–2050 từ toàn bài

Kỹ năng được kiểm tra:
● Đọc thông tin chi tiết      ● Đọc ý chính
● Hiểu ý kiến tác giả         ● Suy luận từ ngữ cảnh
● Đoán nghĩa từ vựng

Chiến lược:
1. Skim → nắm ý chính toàn bài
2. Đọc câu hỏi → gạch từ khóa
3. Scan → tìm phần liên quan
4. Đọc kỹ → loại trừ đáp án sai
```

## SƠ ĐỒ 5 — Cấu trúc chi tiết kỹ năng Nói

```
SPEAKING — 12 phút · 3 phần
Thi trực tiếp với giám khảo (ghi âm toàn bộ)

┌─────────────────────────────────────────────────────────────┐
│ PHẦN 1 — TƯƠNG TÁC XÃ HỘI (Social Interaction)            │
│ Thời gian: ~3 phút                                         │
│                                                             │
│ Giám khảo hỏi → Thí sinh trả lời                           │
│ 2 chủ đề quen thuộc, mỗi chủ đề 3–4 câu hỏi               │
│                                                             │
│ Chủ đề thường gặp (từ 20 đề thật):                         │
│   Travelling · Music · Food · Hobbies · Hometown           │
│   Festivals · Shopping · Art · Daily routine               │
│   Celebrities · Advertisement · Marriage                   │
│                                                             │
│ Cấu trúc mỗi câu trả lời:                                  │
│   Ý chính + Lý do + Ví dụ cá nhân                         │
│   (KHÔNG trả lời Yes/No đơn thuần)                         │
├─────────────────────────────────────────────────────────────┤
│ PHẦN 2 — THẢO LUẬN GIẢI PHÁP (Solution Discussion)        │
│ Thời gian: 1 phút chuẩn bị + 2 phút nói                   │
│                                                             │
│ 1 tình huống → 3 lựa chọn → Chọn 1 và thuyết phục         │
│                                                             │
│ Tình huống hay gặp:                                         │
│   · Địa điểm tổ chức tiệc farewell cho giáo viên           │
│     (trường / nhà GV / nhà hàng)                           │
│   · Cách học tiếng Anh                                     │
│     (online / lớp học / câu lạc bộ)                        │
│   · Quà tặng mẹ nhân sinh nhật                             │
│     (hoa / thiệp handmade / sổ tay)                        │
│   · Địa điểm ôn thi                                        │
│     (nhà / thư viện / quán cà phê)                         │
│   · Cách tìm hiểu lịch sử Việt Nam                         │
│     (bảo tàng / phim tài liệu / sách)                      │
│                                                             │
│ Cấu trúc trả lời:                                          │
│   "I would choose [A] because...                           │
│    Firstly, [lý do 1]...                                   │
│    Moreover, [lý do 2]...                                   │
│    As for [B] and [C], I don't think they're suitable      │
│    because [lý do ngắn gọn]."                              │
│                                                             │
│ ✓ Không có đáp án đúng sai — lập luận tốt là đủ           │
├─────────────────────────────────────────────────────────────┤
│ PHẦN 3 — PHÁT TRIỂN CHỦ ĐỀ (Topic Development)            │
│ Thời gian: 1 phút chuẩn bị + 2–3 phút nói                 │
│           + 3–4 câu hỏi mở rộng từ giám khảo              │
│                                                             │
│ 1 chủ đề + 3 gợi ý ý chính → Trình bày quan điểm         │
│                                                             │
│ Chủ đề hay gặp (từ 20 đề thật):                            │
│   · Lợi ích của Internet / Wifi miễn phí ở trường         │
│   · Học ngoại ngữ là cần thiết                             │
│   · Du học nước ngoài                                      │
│   · Gap year ngày càng phổ biến                            │
│   · Cấm thuốc lá nơi công cộng                             │
│   · Sống xa nhà có nhiều bất lợi                           │
│   · Mua sắm online có nhiều nhược điểm                     │
│   · Đại học không phải con đường duy nhất                  │
│   · Người nên tập yoga                                     │
│   · Đi máy bay khi du lịch                                 │
│   · Nuôi thú cưng dạy trẻ nhiều điều                      │
│   · Lợi ích học đạp xe                                     │
│                                                             │
│ Cấu trúc bài nói 2–3 phút:                                 │
│   Mở:    "Today I'd like to talk about [topic]."           │
│   Ý 1:   Gợi ý 1 + lý do + ví dụ (~40 giây)              │
│   Ý 2:   Gợi ý 2 + lý do + ví dụ (~40 giây)              │
│   Ý 3:   Gợi ý 3 + lý do + ví dụ (~40 giây)              │
│   Kết:   "In conclusion, I strongly believe that..."      │
└─────────────────────────────────────────────────────────────┘
```

## SƠ ĐỒ 6 — Tiêu chí chấm điểm Speaking

```
5 TIÊU CHÍ CHẤM SPEAKING (trọng số ngang nhau)

┌──────────────────────┬──────────────────────────────────────────┐
│ Tiêu chí             │ Giám khảo đánh giá                       │
├──────────────────────┼──────────────────────────────────────────┤
│ 1. VOCABULARY        │ • Từ vựng phù hợp chủ đề                │
│    (Từ vựng)         │ • Đa dạng, có từ nâng cao/hiếm gặp      │
│                      │ • Không lặp từ quá nhiều                 │
│                      │ • Dùng collocation tự nhiên              │
├──────────────────────┼──────────────────────────────────────────┤
│ 2. GRAMMAR           │ • Câu đúng ngữ pháp                      │
│    (Ngữ pháp)        │ • Đa dạng cấu trúc câu                  │
│                      │ • Mix câu đơn + câu phức + câu ĐK        │
│                      │ • Ít lỗi mạo từ, giới từ                │
├──────────────────────┼──────────────────────────────────────────┤
│ 3. PRONUNCIATION     │ • Phát âm đúng âm đơn, âm cuối          │
│    (Phát âm)         │ • Trọng âm từ và câu đúng                │
│                      │ • Ngữ điệu tự nhiên                      │
│                      │ • Không cần giọng bản ngữ                │
├──────────────────────┼──────────────────────────────────────────┤
│ 4. FLUENCY &         │ • Nói liên tục, ít ngập ngừng            │
│    DEVELOPMENT       │ • Biết phát triển và mở rộng ý          │
│    (Trôi chảy)       │ • Sử dụng filler thông minh              │
│                      │ • Không dừng quá lâu                     │
├──────────────────────┼──────────────────────────────────────────┤
│ 5. COHERENCE &       │ • Các ý phù hợp câu hỏi                  │
│    CONTENT           │ • Tổ chức logic, có mở-thân-kết          │
│    (Liên kết ý)      │ • Dùng từ nối rõ ràng                   │
│                      │ • Có ví dụ/giải thích cụ thể             │
└──────────────────────┴──────────────────────────────────────────┘
```

## SƠ ĐỒ 7 — Cấu trúc chi tiết kỹ năng Viết

```
WRITING — 60 phút · 2 bài
Bài thi hoàn toàn trên máy tính

┌──────────────────────────────┬──────────────────────────────────┐
│       TASK 1 (20 phút)       │         TASK 2 (40 phút)         │
│       Chiếm 1/3 điểm         │         Chiếm 2/3 điểm           │
├──────────────────────────────┼──────────────────────────────────┤
│ Viết thư / Email             │ Viết luận (Essay)                │
│ Tối thiểu 120 từ             │ Tối thiểu 250 từ                 │
├──────────────────────────────┼──────────────────────────────────┤
│ Informal letter              │ Discuss both views               │
│ → Bạn bè, người thân         │ → Nêu 2 luận điểm + ý kiến      │
│                              │                                   │
│ Formal letter / Email        │ Opinion essay                    │
│ → Khách sạn, công ty         │ → Đồng ý/Không đồng ý + lý do   │
│   trung tâm, tổ chức         │                                   │
│                              │ Causes & Effects                 │
│ Đề luôn cho sẵn 2–3 ý        │ → Nguyên nhân + Hậu quả         │
│ cần đề cập trong thư         │                                   │
│                              │ Problems & Solutions             │
│ PHẢI đủ tất cả ý yêu cầu!   │ → Vấn đề + Giải pháp            │
│                              │                                   │
│                              │ Advantages & Disadvantages       │
│                              │ → Lợi ích + Hạn chế             │
└──────────────────────────────┴──────────────────────────────────┘

PHẦN "SUGGESTED" TRONG ĐỀ = XƯƠNG SỐNG OUTLINE → Tận dụng tối đa!
```

```
│                        │ • Dấu câu chính xác                          │

⚠ THIẾU 1 TIÊU CHÍ = MẤT ~25% ĐIỂM WRITING
   Lỗi thường gặp nhất: quên dùng linking words (Organization)
```

## SƠ ĐỒ 9 — 4 Dạng Essay và Cấu trúc bài viết

```
DẠNG 1 — DISCUSS BOTH VIEWS + GIVE YOUR OPINION
Nhận diện: "Discuss both views and give your opinion"

┌─────────────────────────────────────────────────────────────┐
│ INTRO      │ Giới thiệu vấn đề + nêu thesis (đứng về bên nào│
│            │ hoặc cân bằng cả 2)                            │
├─────────────────────────────────────────────────────────────┤
│ BODY 1     │ View 1: Luận điểm + Lý do + Ví dụ cụ thể      │
├─────────────────────────────────────────────────────────────┤
│ BODY 2     │ View 2: Luận điểm + Lý do + Ví dụ cụ thể      │
├─────────────────────────────────────────────────────────────┤
│ CONCLUSION │ Tóm tắt + Quan điểm cá nhân rõ ràng           │
└─────────────────────────────────────────────────────────────┘
Đề thật: Athletes' salary vs Teachers' salary (Test 1)
         Watching TV — good or bad for children? (Test 4)
         Films at cinema vs on phone/tablet (Test 10)
         Soft skills vs Technical skills (Test 12)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

DẠNG 2 — OPINION / TO WHAT EXTENT DO YOU AGREE?
Nhận diện: "Give your opinion" / "To what extent do you agree?"

┌─────────────────────────────────────────────────────────────┐
│ INTRO      │ Giới thiệu + nêu rõ quan điểm (đồng ý/không)  │
├─────────────────────────────────────────────────────────────┤
│ BODY 1     │ Luận điểm 1 + Lý do + Ví dụ                   │
├─────────────────────────────────────────────────────────────┤
│ BODY 2     │ Luận điểm 2 + Lý do + Ví dụ                   │
├─────────────────────────────────────────────────────────────┤
│ CONCLUSION │ Nhắc lại quan điểm + Tóm tắt                  │
└─────────────────────────────────────────────────────────────┘
Đề thật: Computers made life complex & stressful (Test 6)
         Teachers not necessary in future (Test 11)
         New experiences — advantages (Test 5)
         Benefits of unpaid work for teenagers (Test 14)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

DẠNG 3 — CAUSES & EFFECTS / PROBLEMS & SOLUTIONS
Nhận diện: "Describe problems... suggest solutions"
           "Why is this trend happening? What are its impacts?"

┌─────────────────────────────────────────────────────────────┐
│ INTRO      │ Giới thiệu vấn đề + phạm vi bài               │
├─────────────────────────────────────────────────────────────┤
│ BODY 1     │ 2–3 Nguyên nhân / Vấn đề + ví dụ cụ thể      │
├─────────────────────────────────────────────────────────────┤
│ BODY 2     │ 2–3 Hậu quả / Giải pháp + ví dụ cụ thể       │
├─────────────────────────────────────────────────────────────┤
│ CONCLUSION │ Tóm tắt vấn đề + kêu gọi hành động (nếu có)  │
└─────────────────────────────────────────────────────────────┘
Đề thật: Overpopulation — problems & solutions (Test 2)
         Obesity — why & impacts (Test 3)
         Causes & effects of stress (Test 16)
```

## Tiêu chí chấm điểm — Bảng chi tiết

### Speaking — 5 tiêu chí

| Tiêu chí | Điểm cao | Điểm thấp |
|----------|----------|-----------|
| **Vocabulary** | Từ vựng phong phú, có từ nâng cao, dùng collocation tự nhiên | Từ vựng nghèo nàn, lặp lại, không phù hợp chủ đề |
| **Grammar** | Đa dạng cấu trúc câu, ít lỗi, dùng câu phức linh hoạt | Chỉ dùng câu đơn, nhiều lỗi ngữ pháp cơ bản |
| **Pronunciation** | Phát âm rõ ràng, trọng âm đúng, ngữ điệu tự nhiên | Phát âm sai nhiều, nghe khó hiểu |
| **Fluency** | Nói trôi chảy, phát triển ý tốt, ít ngập ngừng | Ngập ngừng nhiều, không biết phát triển ý |
| **Coherence** | Ý phù hợp câu hỏi, logic, có từ nối rõ ràng | Lạc đề, ý rời rạc, thiếu kết nối |

### Writing — 4 tiêu chí

| Tiêu chí | Điểm cao | Điểm thấp |
|----------|----------|-----------|
| **Task Fulfillment** | Đủ ý, đúng dạng, đủ từ, không lạc đề | Thiếu ý, sai dạng bài, quá ngắn |
| **Organization** | Bố cục rõ, đoạn liên kết chặt, từ nối phù hợp | Không có bố cục, ý rời rạc, thiếu từ nối |
| **Vocabulary** | Đa dạng, có từ nâng cao, dùng synonyms | Từ nghèo nàn, lặp từ, sai collocations |
| **Grammar** | Cấu trúc đa dạng, ít lỗi, dấu câu đúng | Nhiều lỗi ngữ pháp, chỉ câu đơn đơn giản |

## Từ nối quan trọng cần học cho Writing & Speaking

### Thêm ý / Liệt kê
`Firstly` · `First of all` · `Secondly` · `Thirdly` · `Furthermore` · `Moreover` · `In addition` · `Besides` · `Also` · `Last but not least`

### Đối lập / Tương phản
`However` · `Nevertheless` · `Nonetheless` · `On the other hand` · `In contrast` · `Conversely` · `Although` · `Even though` · `Despite` · `In spite of`

### Nguyên nhân
`Because` · `Since` · `As` · `Due to` · `Owing to` · `As a result of` · `The reason for this is`

### Kết quả / Hậu quả
`Therefore` · `Thus` · `Hence` · `As a result` · `Consequently` · `This leads to` · `This results in`

### Ví dụ / Minh họa
`For example` · `For instance` · `Such as` · `To illustrate` · `A case in point is` · `Including` · `Namely`

### Nhấn mạnh
`In particular` · `Especially` · `Significantly` · `Notably` · `Above all` · `Most importantly`

### Kết luận
`In conclusion` · `To conclude` · `To sum up` · `In summary` · `Overall` · `In short` · `All in all`

## Cách tính điểm chi tiết từng kỹ năng

### Listening & Reading

```
Listening:  Điểm = (Số câu đúng / 35) × 10
Reading:    Điểm = (Số câu đúng / 40) × 10
            Mỗi câu đúng = 0.25 điểm (Reading)
            Làm tròn đến 0.5

Ví dụ Listening: 21/35 đúng → (21/35)×10 = 6.0
Ví dụ Reading:   31/40 đúng → (31/40)×10 = 7.75 → làm tròn 8.0
```

### Writing & Speaking

```
Writing:   Giám khảo chấm trực tiếp theo 4 tiêu chí / thang 10
           Task 1 chiếm 1/3, Task 2 chiếm 2/3 điểm kỹ năng Viết

Speaking:  Giám khảo chấm theo 5 tiêu chí / thang 10
           Dựa trên toàn bộ 3 phần thi ghi âm
           Không có điểm cụ thể cho từng phần
```

### Điểm tổng & Quy bậc

```
Điểm tổng = (Listening + Reading + Writing + Speaking) / 4
Làm tròn đến 0.5

Quy bậc:
  4.0 – 5.5  →  B1 (Bậc 3)
  6.0 – 8.0  →  B2 (Bậc 4)
  8.5 – 10.0 →  C1 (Bậc 5)
```

## Tổng hợp lỗi thường gặp & Cách khắc phục

### Speaking

| Lỗi thường gặp | Cách khắc phục |
|----------------|----------------|
| Trả lời Yes/No rồi dừng | Luôn thêm lý do + ví dụ cá nhân |
| Part 2: chỉ nói về lựa chọn mình chọn | Phải giải thích tại sao KHÔNG chọn 2 cái còn lại |
| Part 3: hết ý sớm, im lặng | Dùng câu chuyển: "In addition, I'd also like to mention..." |
| Phát âm không rõ âm cuối | Luyện âm /t/ /d/ /s/ /z/ /ʃ/ ở cuối từ |
| Lặp từ "because" và "also" | Học từ nối đa dạng: moreover, furthermore, additionally |

### Writing

| Lỗi thường gặp | Cách khắc phục |
|----------------|----------------|
| Không xác định được dạng đề | Đọc câu cuối đề — thường ghi rõ yêu cầu |
| Bỏ qua phần "Suggested" trong đề | Đây là gợi ý chính thức — dùng làm outline |
| Không đủ 250 từ Task 2 | Đếm từ sau khi viết, thêm ví dụ/giải thích |
| Thiếu linking words | Gạch chân từ nối trong bài sau khi viết |
| Mở bài chép lại đề | Paraphrase câu hỏi bằng từ của mình |
| Kết bài quá ngắn | Kết bài phải: tóm tắt + nêu lại quan điểm (2–3 câu) |

## Checklist hoàn chỉnh trước ngày thi

### Nghe (Listening)
- [ ] Luyện đọc câu hỏi nhanh trước khi nghe
- [ ] Luyện gạch chân từ khóa trong câu hỏi
- [ ] Luyện nghe học thuật (TED, BBC, podcast)
- [ ] Làm quen với tốc độ nghe của bài giảng

### Đọc (Reading)
- [ ] Luyện kỹ thuật skimming (đọc lướt nhanh)
- [ ] Luyện kỹ thuật scanning (tìm thông tin cụ thể)
- [ ] Luyện loại trừ đáp án sai
- [ ] Không dành quá 15 phút cho 1 bài đọc

### Nói (Speaking)
- [ ] Luyện cấu trúc: ý chính + lý do + ví dụ cho mọi câu hỏi
- [ ] Luyện Part 2: chọn 1/3 + so sánh + từ chối 2 cái còn lại
- [ ] Luyện nói 2–3 phút liên tục không dừng Part 3
- [ ] Học thuộc các câu chuyển ý, câu mở, câu kết
- [ ] Luyện phát âm âm cuối rõ ràng

### Viết (Writing)
- [ ] Nhận diện 4 dạng đề trong 30 giây đầu
- [ ] Luyện viết outline trước (2–3 phút)
- [ ] Task 1: đảm bảo ≥ 120 từ, đủ tất cả ý yêu cầu
- [ ] Task 2: đảm bảo ≥ 250 từ, đủ bố cục Intro-Body-Conclusion
- [ ] Đọc lại bài và sửa lỗi ngữ pháp/chính tả (dành 3–5 phút)
- [ ] Tận dụng phần "Suggested" làm xương sống outline
