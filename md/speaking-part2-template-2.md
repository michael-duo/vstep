# SPEAKING PART 2 — TEMPLATE & HƯỚNG DẪN

## Dạng bài

**Situation**: tình huống, 3 options (3 lựa chọn) → Phù hợp nhất? Tại sao?

- 1 phút chuẩn bị — take note
- 3 phút speaking

---

## Template chính

### Mở đầu (học thuộc lòng — dùng N / Ving)

### Thân

- **Tôi chọn A vì có nhiều lợi ích**
  - 2–3 lý do (có ý phụ)
  - First / To begin with / When it comes to, ... For example / It means that / ...
  - Second / One more point worth considering is ...
  - Last / Last but not least / The last reason that makes me choose A is ...

- **Tôi không chọn B vì (1 lý do)**
  - B is not a reasonable option because ...

- **Tôi không chọn C vì (1 lý do)**

### Kết (học thuộc)

---

## Tận dụng 1 phút chuẩn bị

**Ở nhà:**
- Học thuộc Template
- Học thuộc Ideas cho 5 dạng

**Đi thi:**
- A, B, C → xác định dạng → Chọn ra lựa chọn nào đạt được 2–3 tiêu chí ideas đã chuẩn bị
- Lắp vào template

---

## Template 2 (chi tiết)

| Phần | Nội dung |
|------|----------|
| **Câu mở đầu** | If I had to choose among A, B and C to + *mục đích (đổi bài en paraphrase lại)*, I would definitely go for A for several reasons. |
| **Chứng minh A là lựa chọn tốt hơn** | Compared to B and C, A has many advantages. *(kể các lợi ích của A)* |
| | **Lưu ý:** sử dụng linh hoạt các cụm từ để giải thích và đưa ra ví dụ: |
| | • elaborate on / for example / for instance |
| | • Nhấn mạnh thêm: *Only a/an ... a chance to ...* |
| | • *A is an ideal option because it gives me a chance to ...* |
| | • *Thanks to A, I can ...* |
| **Chứng minh A tốt hơn B, C** | Dùng các cấu trúc so sánh: |
| | • So sánh hơn / so sánh nhất |
| | • Câu ghép: although / while / despite ... |
| | • Liệt kê: not only ... but also ... |
| **Chứng minh B và C ít tốt hơn** | Meanwhile, regarding / concerning B and C, *(nêu mặt bất lợi của B và C)* |
| **Kết luận** | For all these reasons, A is the best option. |

## Nâng cấp lên B2

- Thêm cấu trúc câu **bị động** (S + be + V-pp)
- Dùng thì **hiện tại hoàn thành**
- Cấu trúc câu **ghép / phức**: If, so sánh, đảo ngữ
- Ưu tiên dùng **cụm danh từ**:

| Cách nói đơn giản | Nâng cấp |
|-------------------|----------|
| it is cheap / expensive | reasonable / affordable |
| | offer a low price / high price / over price |
| decide to V | make a decision |
| order | make an order |
| reserve | make a reservation |
