## Từ chỉ dẫn trong câu hỏi — Phải biết!

Đây là những từ xuất hiện trong **câu hỏi**, không phải trong bài đọc. Hiểu sai từ chỉ dẫn = trả lời sai dạng.

| Từ chỉ dẫn | Nghĩa | Dạng câu hỏi |
|-----------|-------|-------------|
| **according to** | theo (bài viết) | Thông tin cụ thể |
| **mention / state** | đề cập / nêu | Thông tin cụ thể |
| **mainly about** | chủ yếu về | Ý chính |
| **main purpose** | mục đích chính | Ý chính |
| **most closely means** | gần nghĩa nhất với | Từ vựng |
| **as used in** | như được dùng trong | Từ vựng |
| **infer / imply** | suy ra / ngụ ý | Suy luận |
| **suggest** | gợi ý, cho thấy | Suy luận |
| **refer to** | chỉ đến, ám chỉ | Từ vựng / Suy luận |
| **EXCEPT / NOT** | ngoại trừ / không | Loại trừ (cẩn thận!) |

## Từ vựng phổ biến trong bài đọc B1

### Nhóm 1: Xã hội & Đời sống

| Từ | Nghĩa | Ví dụ ngữ cảnh |
|---|-------|---------------|
| **community** | cộng đồng | *The local community organized a festival.* |
| **population** | dân số | *The population of this city is growing.* |
| **influence** | ảnh hưởng | *Social media has a strong influence on young people.* |
| **benefit** | lợi ích | *Regular exercise has many health benefits.* |
| **issue / problem** | vấn đề | *Pollution is a serious issue in big cities.* |
| **opportunity** | cơ hội | *Education provides more job opportunities.* |
| **increase / decrease** | tăng / giảm | *The number of tourists has increased significantly.* |
| **improve** | cải thiện | *The government wants to improve public transportation.* |

### Nhóm 2: Khoa học & Công nghệ

| Từ | Nghĩa | Ví dụ ngữ cảnh |
|---|-------|---------------|
| **research / study** | nghiên cứu | *A recent study shows that...* |
| **develop** | phát triển | *Scientists are developing new technology.* |
| **significant** | đáng kể, quan trọng | *There has been a significant change.* |
| **evidence** | bằng chứng | *There is strong evidence that...* |
| **method / approach** | phương pháp | *This method is more effective.* |
| **effective** | hiệu quả | *Online learning can be very effective.* |
| **discover** | phát hiện | *Researchers discovered a new species.* |
| **process** | quy trình | *The learning process takes time.* |

### Nhóm 3: Môi trường

| Từ | Nghĩa | Ví dụ ngữ cảnh |
|---|-------|---------------|
| **pollution** | ô nhiễm | *Air pollution affects people's health.* |
| **climate** | khí hậu | *Climate change is a global problem.* |
| **protect** | bảo vệ | *We should protect the environment.* |
| **sustainable** | bền vững | *Sustainable development is important.* |
| **resource** | tài nguyên | *Water is a valuable resource.* |
| **reduce** | giảm | *We need to reduce waste.* |

## Từ nối trong bài đọc — giúp hiểu cấu trúc

| Từ nối | Nghĩa | Báo hiệu |
|--------|-------|----------|
| **however / nevertheless** | tuy nhiên | Ý đối lập sắp đến |
| **furthermore / moreover** | hơn nữa | Thêm ý bổ sung |
| **therefore / as a result** | vì vậy / kết quả là | Kết luận |
| **in contrast** | ngược lại | So sánh đối lập |
| **for example / for instance** | ví dụ | Minh họa |
| **in other words** | nói cách khác | Giải thích lại |
| **despite / in spite of** | mặc dù | Ý trái ngược |
