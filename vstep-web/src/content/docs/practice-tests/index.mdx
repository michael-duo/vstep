Trang này tổng hợp **20 bộ đề thi thật** từ các trường Đại học Ngoại ngữ và Đại học Hà Nội (2019-2021), có **hướng dẫn giải mẫu** theo cách đơn giản nhất cho trình độ B1.

> Mỗi bộ đề gồm **Speaking (Part 1-3)** và **Writing (Task 1-2)** với đầy đủ: đáp án gợi ý + giải thích cấu trúc + từ vựng quan trọng.

**Cách dùng:** Chọn đề bất kỳ → đọc đề → đọc phân tích → tự luyện nói/viết theo dàn ý.

## Chủ đề Speaking hay gặp nhất

Dựa trên 20 đề trên, các chủ đề **xuất hiện nhiều lần** (ưu tiên ôn trước):

| Chủ đề | Số lần xuất hiện |
|---|---|
| **Hometown / Where you live** | 3 lần (Đề 7, 18, + liên quan) |
| **Education / Study / English** | 3 lần (Đề 1, 11, 20) |
| **Food / Cooking / Health** | 3 lần (Đề 3, 5, 15) |
| **Travel / Holidays** | 3 lần (Đề 2, 5, 8) |
| **Hobbies / Free time** | 2 lần (Đề 17, 16) |
| **Family / Relationships** | 2 lần (Đề 6, 12) |
| **Animals / Pets** | 2 lần (Đề 6, 13) |
| **Technology / Internet** | 2 lần (Đề 4, 17) |

## Dạng Writing hay gặp nhất

| Dạng bài | Số lần | Đề tham khảo |
|---|---|---|
| **Discuss both views + opinion** | 7 lần | Đề 1, 4, 10, 12, 15, 18, 19 |
| **Informal letter** (viết cho bạn) | 12 lần | Đề 1, 2, 5, 7, 8, 13, 14, 15, 17, 19, 20 |
| **Formal letter** (viết cho tổ chức) | 5 lần | Đề 3, 9, 11, 12, 16 |
| **Causes + Effects** | 3 lần | Đề 3, 16 |
| **Advantages / Benefits** | 3 lần | Đề 5, 14 |
| **Benefits + Drawbacks** | 2 lần | Đề 13, 17 |
